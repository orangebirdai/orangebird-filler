## Deploy in 60 seconds
1. Click "Use this template" → Create new repo in your account
2. Go to Render.com → New Web Service → connect your repo
3. Start command: `uvicorn app.py:app --host 0.0.0.0 --port $PORT`
4. Add env var `GROQ_API_KEY` = your real key
5. Deploy → done!
